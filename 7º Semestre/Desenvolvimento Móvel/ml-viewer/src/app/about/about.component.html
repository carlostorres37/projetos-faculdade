Sobre

Página sobre o trabalho.

O trabalho consiste em realizar uma busca por nome do produto e retornar uma lista com 50 produtos.

Esse retorno é realizado através de uma requisição à API de busca do MercadoLivre. O endpoint específico para busca retorna 50 produtos que coincidem com o termo pesquisado.

Essa página serve propositalmente para a exemplificação de rotas na aplicação
